# @scout-mobile/framework-rn

React Native framework adapter for [Scout](https://github.com/nicholaskasten/scout-mobile). Provides React Native-specific inspection and Metro bundler integration.

## Install

```bash
npm install @scout-mobile/framework-rn
```

## Requirements

- Node.js >= 20
- A React Native project

## Usage

This package is used internally by `@scout-mobile/core`. You generally don't need to install it directly unless building a custom adapter.

```typescript
import { ReactNativeFrameworkAdapter } from '@scout-mobile/framework-rn';
```
